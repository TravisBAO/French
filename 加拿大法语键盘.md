Windows 的系统里的加拿大法语键盘，特殊字符的输入法如下，供大家参考

*说明：/ 表示同时按，+ 表示前面松开后按  

- **é = 按 /** 

- **à, è, ì, ò, ù =＇+ a, e, i, o, u**
  
- **â, ê, î, ô, û = [ + a, e, i, o, u**  
  
- **ç = ] + c**  
  
- **ä, ë, ï, ö, ü = Shift /} /Caps + a, e, i, o, u**  
  
- **œ = ctrl /shift/7 + o**  
  
- **€ = ctrl /alt /e**
