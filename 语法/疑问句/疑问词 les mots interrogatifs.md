下面是**法语常见疑问词（les mots interrogatifs）**的汇总表，按含义分类，并附带例句。

## 📘 法语疑问词总览表

| 疑问词   | 中文意思      | 发音（IPA）     | 用法/说明    | 例句     |
| ------ | --------- | ------ | -------- | -------- |
| **Qui**      | 谁         | \[ki]                                         | 指人，作主语或宾语                   | **Qui** est-ce ?（这是谁？）                 |
| **Que / Qu'**                                 | 什么        | \[kə] / \[k‿]                                 | 作宾语，主语时用 **Qu’est-ce qui…** | **Que** fais-tu ?（你在做什么？）              |
| **Quoi**                                      | 什么        | \[kwa]                                        | 用于介词之后或句尾                   | Tu parles de **quoi** ?（你在说什么？）        |
| **Quel / Quelle / Quels / Quelles**           | 哪一个？什么样的？ | \[kɛl] / \[kɛl] / \[kɛl] / \[kɛl]             | 随性数变化                       | **Quelle** heure est-il ?（现在几点？）       |
| **Où**                                        | 哪里        | \[u]                                          | 问地点                         | Tu vas **où** ?（你去哪？）                  |
| **D’où**                                      | 从哪里       | \[du]                                         | 问来源                         | Tu viens **d’où** ?（你从哪来？）             |
| **Quand**                                     | 什么时候      | \[kɑ̃]                                        | 问时间                         | **Quand** part-il ?（他什么时候出发？）          |
| **Comment**                                   | 怎样，如何     | \[kɔ.mɑ̃]                                     | 问方式或状态                      | **Comment** ça va ?（你好吗？）              |
| **Combien (de)**                              | 多少        | \[kɔ̃.bjɛ̃]                                   | **de + 名词**时表示数量            | **Combien de** livres as-tu ?（你有多少本书？） |
| **Pourquoi**                                  | 为什么       | \[puʁ.kwa]                                    | 问原因                         | **Pourquoi** pleures-tu ?（你为什么哭？）      |
| **Parce que**                                 | 因为        | \[paʁs‿kə]                                    | 回答 *Pourquoi* 问题            | Parce que je suis fatigué.             |
| **Lequel / Laquelle / Lesquels / Lesquelles** | 哪一个（代词）   | \[lə.kɛl] / \[la.kɛl] / \[le.kɛl] / \[lɛ.kɛl] | 指代前面提到的名词                   | **Lequel** préfères-tu ?（你更喜欢哪一个？）     |
