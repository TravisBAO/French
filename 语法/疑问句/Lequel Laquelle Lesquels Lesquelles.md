## 📘 1. 基本含义

lequel / laquelle / lesquels / lesquelles
都是 关系代词或疑问代词，意思是：

- 👉 “哪一个 / 哪些”
- 👉 “which one / which (ones)”

它们必须与它们所指代的 名词的性别和数 一致。

## 📘 2. 形式总览（带发音）
| 形式             | 发音（IPA）  | 性 / 数 | 中文意义 | 英文意义       |
| -------------- | -------- | ----- | ---- | ---------- |
| **lequel**     | [lə.kɛl] | 阳性单数  | 哪一个  | which one  |
| **laquelle**   | [la.kɛl] | 阴性单数  | 哪一个  | which one  |
| **lesquels**   | [le.kɛl] | 阳性复数  | 哪些   | which ones |
| **lesquelles** | [lɛ.kɛl] | 阴性复数  | 哪些   | which ones |
## 📘 3. 用法一：疑问代词（问句）
✔ 用来问“哪一个？哪些？”

| 法语句子                         | 中文            | 英文                         |
| ---------------------------- | ------------- | -------------------------- |
| **Lequel veux-tu ?**         | 你想要哪一个？       | Which one do you want?     |
| **Laquelle préfères-tu ?**   | 你更喜欢哪一个（阴性）？  | Which one do you prefer?   |
| **Lesquels as-tu choisis ?** | 你选了哪几个（阳性复数）？ | Which ones did you choose? |
| **Lesquelles sont à toi ?**  | 哪些是你的（阴性复数）？  | Which ones are yours?      |

## 📘 4. 用法二：关系代词（指代先行词）

**用于替代 le + 名词、la + 名词、les + 名词**。

英文对应：

“which” / “whom” / “that”（取决于结构）

✔ 例句
| 法语句子                                            | 中文          | 英文                                   |
| ----------------------------------------------- | ----------- | ------------------------------------ |
| **Le livre dans lequel j’ai écrit…**            | 我在其中写东西的那本书 | the book **in which** I wrote        |
| **La ville à laquelle je pense…**               | 我正在想的那个城市   | the city **that** I’m thinking about |
| **Les personnes avec lesquelles je travaille…** | 我一起工作的那些人   | the people **with whom** I work      |
| **Les idées parmi lesquelles j’ai choisi…**     | 我从中挑选出的那些想法 | the ideas **among which** I chose    |

## 📘 5. 与介词结合时的变化

| 介词              | 阳性单数       | 阴性单数        | 阳性复数     | 阴性复数       |
| --------------- | ---------- | ----------- | -------- | ---------- |
| **à + lequel**  | **auquel** | à laquelle  | auxquels | auxquelles |
| **de + lequel** | **duquel** | de laquelle | desquels | desquelles |

### ✔ 常用结构示例

auquel je pense（我想到的……）

duquel il parle（他谈到的……）

auxquels elle répond（她回答的那些……）

desquelles tu viens（你来自的那些……）

## 📘 6. 小总结（超级清晰版）
### A. 疑问：

 - Lequel？哪个（阳性）

 - Laquelle？哪个（阴性）

 - Lesquels？哪些（阳性复数）

 - Lesquelles？哪些（阴性复数）

### B. 关系代词：

 - 代替 “le/la/les + 名词”

 - 与介词连用常变成：auquel / duquel / auxquels / etc.
